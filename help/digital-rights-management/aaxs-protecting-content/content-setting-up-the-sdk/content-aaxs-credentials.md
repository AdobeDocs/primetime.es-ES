# Credenciales de acceso al Adobe{#adobe-access-credentials}

Se requieren 4 tipos de credenciales para utilizar Acceso a Adobe:

* **Empaquetador**: se utiliza durante el empaquetado para firmar los metadatos a√±adidos al contenido cifrado
* **Servidor** de licencias: se utiliza para proteger la clave de cifrado de contenido en los metadatos y el servidor de licencias la utiliza para firmar licencias.
* **Transporte:** se utiliza para proteger las solicitudes/respuestas intercambiadas entre el cliente y el servidor de licencias.
* **Dominio CA**: se usa para emitir certificados de dominio a dispositivos que se unen a un dominio.
